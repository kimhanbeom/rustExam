1.Comments
------------------

- rust의 Comments는 C와 같다.
  1) // (한줄)
  ```rust
    fn main() {
      //한줄 comment!!!
      //한줄 comment!!!
      let some_number = 100;
    }
  ```
  2) /* */ (범위 block)
  ```rust
    fn main() {
      /*
       범위 comment!!!
       범위 comment!!!
       범위 comment!!!
      */
      let some_number = 100;
    }
  ```
