1.선언
------
  - fn으로 시작한다.
  - 다른언어와 마찬가지로 in(매개변수)/out(리턴값) 형식으로 작성
  ```rust
  fn another_function(x:i32) -> i32 {
        let sum = x + x;
        x
  }
  ```

2.구문과 표현식
-----
  - 구문 : 명령들의 나열, 값을 반환하지 않는 어떠한 동작을 수행
   ```rust
   let y = 6;
   ```
  - 표현식 : 결과 값을 산출
   ```rust
       x+1
   ```
   ```rust
      { let x =3;  x+1 }  //block의 산출물은 x+1이고 이는 표현식
   ```
  - 대부분의 상황에서 표현식에 세미콜론이 들어가면 구문이 됨
